gh-vehicle-duty API

Ghana vehicle import duty · for developers

Calculate Ghana customs duties and levies for imported vehicles. Reproduces the levy schedule of a GRA Bill of Entry, validated line-by-line. Free, open source, MIT licensed.

Writing JavaScript? Use the library instead — no network call, works offline, and you stay pinned to a known rate version.
npm install gh-vehicle-duty

Endpoints

EndpointDescription
POST/calculateDuty breakdown from a JSON body.
GET/calculateSame, via query string.
GET/ratesThe full versioned levy schedule.
GET/healthUptime and rate freshness. Returns 503 when stale.

Check the rate version

  
Ghana's levies change with each budget. Every response carries a staleness block and an
  X-Rates-Staleness header. If it isn't current, warn your users.

  

    Levels: current → aging (365+ days — verify the rates still apply) →
    stale (550+ days — /health returns 503).


    Don't cache results across a budget cycle. For production, deploy your own copy rather than depending on
    someone else's.
  


  
Generate a client

  
An OpenAPI 3.1 spec
  is included, so you can generate a typed client for Java, C#, Swift, Kotlin, Go, or anything else:

  
npx @openapitools/openapi-generator-cli generate \
  -i openapi.yaml -g python -o ./client


  
Run your own

  
Deploying takes about two minutes on Cloudflare's free tier — no command line needed. See
  DEPLOY.md.
  There's no database and no auth; it's pure arithmetic.


  

    Estimate only. This applies a levy schedule to a customs value you supply. It does not know GRA's official
    valuation, which is assessed from the vehicle VIN against GRA's ICUMS/HDV benchmark. Not affiliated with or
    endorsed by the Ghana Revenue Authority. Not a substitute for a licensed clearing agent.
