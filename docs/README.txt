GHANA VEHICLE DUTY CALCULATOR — install as an app
==================================================

This folder is a complete Progressive Web App (PWA). Once it's online,
you can install it on your laptop AND your phone with an icon, and it
works offline.

STEP 1 — Put it online (free, ~2 minutes, no coding)
  Easiest: go to  https://app.netlify.com/drop
    - Drag THIS WHOLE FOLDER (duty-app) onto the page.
    - It gives you a link like  https://something.netlify.app
  (Vercel, Cloudflare Pages, or GitHub Pages also work the same way.)

STEP 2 — Install on your LAPTOP (Chrome or Edge)
  - Open the link.
  - Click the install icon at the right of the address bar
    (a small monitor/⊕ icon), then "Install".
  - It opens in its own window and gets a desktop/Start-menu icon.

STEP 3 — Install on your PHONE
  iPhone (Safari):  open link → Share button → "Add to Home Screen".
  Android (Chrome): open link → menu (⋮) → "Install app" / "Add to Home Screen".

That's it — tap the icon any time, even with no internet.

TIP: To just try it without hosting, double-click index.html to open it
in a browser. (Installing/offline only works from a hosted https link.)
